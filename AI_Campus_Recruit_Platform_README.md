# AI Campus Recruit Platform

## 启动方式

```bash
pnpm dev
```

该命令会同时启动：
- MySQL（3306）与 Redis（6379），优先使用 Docker Compose 自动拉起
- 前端 Vite 开发服务（热加载）
- 后端 FastAPI `uvicorn --reload`（热加载）

## 当前技术栈

- 后端：FastAPI + SQLAlchemy + MySQL + Redis
- 安全：JWT + RBAC + BCrypt 密码哈希
- 前端：Vue3 + Vite

## 项目简介

AI Campus Recruit Platform 是一个 **AI驱动的校园招聘平台**，连接
**学校主管、企业招聘方、学生面试方** 三方，通过 **AI + RAG（Retrieval
Augmented Generation）** 提升招聘效率与职业学习效率。

平台解决传统校园招聘中的以下问题： - 企业资质不透明 -
学生与岗位匹配困难 - 招聘流程效率低 - 学生缺乏面试准备 - 职业知识碎片化

通过 AI 技术，本平台提供： - AI岗位匹配 - AI模拟面试 - AI面试官 -
职业RAG知识助手

# 系统角色

系统包含三种角色：

## 1 学校主管（Admin）

职责：

-   审核企业资质
-   监管招聘行为
-   核实学生信息
-   管理就业数据

主要功能：

-   企业资质审核
-   学生身份认证
-   招聘活动监管
-   数据统计分析

------------------------------------------------------------------------

## 2 招聘方（企业）

企业必须经过 **学校审核** 才能发布招聘信息。

企业功能：

### 企业主页

企业可以发布：

-   企业介绍
-   企业文化
-   技术栈
-   团队介绍
-   企业宣传视频

### 发布岗位

岗位信息包括：

-   岗位名称
-   岗位职责
-   技能要求
-   薪资范围
-   工作地点
-   学历要求
-   招聘人数
-   截止时间

技能标签示例：

-   Kubernetes
-   Docker
-   Python
-   Linux
-   DevOps
-   SRE
-   Go
-   AI

### 学生管理

企业可以：

-   查看学生简历
-   查看技能标签
-   查看学校认证信息

### 聊天系统

类似 BOSS直聘 的沟通模式

支持：

-   实时聊天
-   文件发送
-   简历发送
-   面试邀请

技术实现：

WebSocket

### AI面试

企业可以创建 AI面试流程。

例如：

SRE岗位AI面试

AI自动生成问题：

-   Kubernetes调度机制
-   Linux内存管理
-   TCP三次握手
-   Prometheus原理

AI自动生成评估报告：

-   技术能力
-   表达能力
-   逻辑能力
-   岗位匹配度

# 学生端功能

## 在线简历

学生可以创建在线简历。

字段包括：

-   姓名
-   学校
-   专业
-   学历
-   技能标签
-   项目经历
-   实习经历
-   证书
-   自我介绍

支持：

-   在线简历
-   PDF简历

------------------------------------------------------------------------

## AI岗位推荐

系统会根据学生简历自动匹配岗位。

流程：

学生简历 → Embedding → 向量搜索 → 岗位匹配

返回：

-   岗位推荐
-   匹配度
-   推荐理由
-   技能差距分析

示例：

岗位：SRE\
匹配度：85%

匹配技能：

-   Linux
-   Docker
-   Kubernetes

建议补充：

-   Prometheus
-   CI/CD

------------------------------------------------------------------------

## 职业RAG助手

学生可以询问职业相关问题。

示例：

-   SRE需要学习什么？
-   DevOps和SRE区别
-   如何成为云原生工程师？

系统会返回：

-   职业知识
-   学习路径
-   技能树
-   推荐课程

示例学习路径：

Linux\
├─ 网络\
├─ 系统\
└─ Shell

云原生\
├─ Docker\
├─ Kubernetes\
└─ Service Mesh

可观测性\
├─ Prometheus\
├─ Grafana\
└─ OpenTelemetry

------------------------------------------------------------------------

## AI模拟面试

学生可以使用 AI 模拟真实面试。

AI会：

-   自动生成问题
-   实时评分
-   给出改进建议
-   提供标准答案

# RAG系统设计

知识来源：

-   企业岗位JD
-   技术文档
-   行业报告
-   面试题库
-   学习资料

RAG流程：

文档 → 分块 → Embedding → 向量数据库 → 检索 → LLM生成回答

向量数据库推荐：

-   Milvus
-   Weaviate
-   pgvector

# AI模块

平台包含四类AI能力：

## 1 岗位匹配AI

输入：学生简历\
输出：推荐岗位

## 2 AI面试官

输入：岗位JD\
输出：面试问题 + 评分

## 3 职业RAG助手

输入：职业问题\
输出：职业知识与学习路径

## 4 AI模拟面试

输入：目标岗位\
输出：模拟面试问题与反馈
